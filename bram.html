BRAM

  • Clock Skew/: the difference in time from when the clock arrives at one flip-flop and another flip-flop.
    • Routing logic is pretty good, but it's not instantaneous.
    • Tools these days now can account for skew in timing analysis.
  • Signals
    • SCLK: clock
    • MOSI: Manager Out, Subordinate In
    • MISO: Manager In, Subordinate Out
      • SDRAM, cameras, special sensors
      • Phase-Locked Loop (PPL) does this: takes in a reference clock, then it will spin it up to a very high rate (GHz-range) then tap off it to divide down to the clock you want
      • Will need multiple clocks for high-speed serial interfaces, high speed memory interfaces
    • CS/SS: Chip Select/Subordinate Select
    • Clock Domain Crossing: when clocks cross from one domain to another
